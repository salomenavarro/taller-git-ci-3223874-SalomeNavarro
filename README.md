# Taller Git CI – SENA

Pequeña aplicación web que permite agregar elementos a una lista con una interfaz básica.  
Este proyecto se usa para practicar:

- Commits atómicos
- Conventional Commits
- Issues
- Branching
- Pull Requests
- Code Review

## Requisitos

- Node.js 18+
- npm

npm install
